# Myco Labs Visual Asset Prompts

**Purpose:** Optional prompts for future generated images or short loops. These are not required for the first fast Three.js website build, but they can help create richer visuals if the site later needs more photographic or cinematic assets.

Keep all assets aligned with the Myco Labs brand: premium dark background, emerald spore glow, Gujarat manufacturer focus, practical B2B credibility, no public AI messaging.

---

## 1. Abstract Hero Background Still

Use for: fallback hero background, Open Graph preview, social sharing image.

```text
Create a premium abstract background for a B2B industrial digital agency called Myco Labs. Deep near-black and navy background, one glowing emerald green circular spore mark, subtle connected network lines spreading outward like a mycelium network, faint industrial grid texture, cinematic soft light, high contrast, elegant and minimal, no text, no people, no logos, no purple gradient, no cartoon style, 16:9 aspect ratio.
```

## 6. Short Looping Hero Video Prompt

Use for: optional compressed MP4/WebM fallback or social clip. Not required for the first website build.

```text
Create a 6-second seamless looping cinematic animation for Myco Labs. A glowing emerald circular spore appears on a deep dark navy background, then expands into a refined network of connected nodes representing manufacturer catalogs, websites, WhatsApp, marketplaces, and Google profiles. Smooth premium motion, subtle industrial grid, slow camera drift, no text, no logos, no people, no aggressive neon, no cyberpunk glitch. The loop should start and end on a similar glowing spore state.
```

Implementation note: if used on the website, export as compressed WebM and MP4, keep under 2 MB, lazy-load below hero or use only as fallback/social media asset. Do not replace the real Three.js hero with heavy video.

---

## 7. First-Build Asset Decision

For the first implementation, prefer procedural Three.js visuals and CSS texture over generated media. Use generated assets only after the core site is fast, responsive, and verified.
