# myco labs Design Direction

## Brand feeling
Premium, clean, calm, modern, trustworthy, local-business friendly.

## Current UI problem
The current website feels too dark, heavy, and not properly aligned. The redesign should make it lighter, cleaner, more spacious, and easier to understand.

## Visual direction
- Use off-white / soft light background as the main base
- Use deep navy or charcoal for text
- Use emerald green as the main accent
- Use soft borders instead of heavy glowing effects
- Use subtle shadows only where useful
- Use large spacing between sections
- Keep cards aligned and consistent
- Make headings clear and readable

## Avoid
- Too much black background
- Too many gradients
- Random glow effects
- Unaligned cards
- Over-animation
- Tiny text
- Too many different font sizes
- Generic AI startup look

## Layout direction
- Clear hero section
- Strong headline
- Simple subheadline
- Visible CTA
- Services section should be easy to scan
- Sections should follow consistent max-width
- Mobile should look clean first

## Preferred CTA
Primary: "Get Free Demo on WhatsApp"
Secondary: "See Our Work"

## Target audience
Small businesses in Gujarat:
- property dealers
- coaching classes
- mobile shops
- clinics
- repair services
- local service providers
